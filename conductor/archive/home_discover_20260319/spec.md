# Specification: Implement Home Discover Screen

## Goal
The goal of this track is to migrate the design and layout of the "Home Discover Screen" from the provided HTML/CSS resources into the live Next.js application, ensuring it is functional, responsive, and follows the project's design guidelines.

## Requirements
- **Layout Migration:** Convert `resources/home_discover_screen/code.html` into React components in `app/page.tsx` or separate component files.
- **Shared Components:** Create reusable `Header` and `Footer` components as seen in the design.
- **UI Kit:** Implement shared UI elements like `Button`, `Input`, and `PropertyCard` following the Nordic Minimalist style.
- **Responsiveness:** Ensure the page works perfectly on mobile, tablet, and desktop.
- **Color Palette:** Use the specified colors: Primary (#06f9d0), Background Light (#EEF6F6), Nordic Dark (#19322F).
- **Icons:** Integrate Material Symbols Outlined.
