# LuxeEstate Tech Stack

## Core Technologies
- **Language:** [TypeScript](https://www.typescriptlang.org/) — Providing strong typing and improved developer experience.
- **Frontend Framework:** [Next.js 16 (App Router)](https://nextjs.org/) — Utilizing React Server Components (RSC) for optimized performance and a modern routing architecture.
- **Library:** [React 19](https://react.dev/) — Leveraging the latest React features for building interactive UIs.
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/) — A utility-first CSS framework for rapid and consistent UI development.

## Development Tools
- **Linting:** [ESLint](https://eslint.org/) — Maintaining code quality and consistency across the project.
- **Type Checking:** [TypeScript](https://www.typescriptlang.org/) — Ensuring type safety and catching errors early in development.

## Asset & Design Integration
- **Iconography:** [Material Symbols Outlined](https://fonts.google.com/icons) — Modern, scalable icons integrated via a suitable React library or direct SVG imports.
- **Prototypes:** [HTML/CSS Designs](./resources/) — Comprehensive design references available in the `resources/` directory for implementation.
